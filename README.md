## 项目介绍
本项目是基于Python Flask框架开发的简单在线购物网站，满足实验要求的顾客端和销售管理端全部功能，数据库采用MySQL，支持在线部署到阿里云ECS。

## 技术栈
- 后端：Python 3.8+、Flask 2.3.3、SQLAlchemy 3.1.1
- 前端：HTML5、CSS3、Bootstrap 5
- 数据库：MySQL 8.0
- 部署：Gunicorn、Nginx、Supervisor
